Saving & Recalling Panel Setups

If a complex panel set-up is required for a particular circuit it is far more efficient for the oscilloscope to remember the configuration of the panel then needing to input the configuration before every start-up. Keep in mind the TAS 455 is from 1992 so this memory solution will probably appear to be archaic.

Saving a Setup

    Press the SAVE/RECALL SETUP button to display the setup menu. Select SAVE and memory location 1, which will place the current setup in memory location one.

Recalling a Setup

After saving a setup you can recall it with the following steps

    Press the SAVE/RECALL SETUP button to display the setup menu. Select RECALL with the menu buttons. Select the location from which you want to retrieve a front panel setup.
